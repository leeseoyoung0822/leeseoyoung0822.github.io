나의 두번째 블로그

deep-learning

Published on May 15, 2023

Reading time: 6 minutes.

분류

: 입력 → 함수 → 출력(강아지) ; 출력이 어떤 클래스에 속하는 걸 찾아낼 수 있는지

→ 0 또는 1로 바꿔서 분류함.

백터 만들어서 “더 핫 인코딩”

  • 정확도(accuracy) : 여러 입력값에 대한 예측치가 실제 label과 얼마나 맞는지(보통 10개 중 1개, 100개일 때 1 %)
  • loss : 내가 학습시키기 위해 scalar를 정하고, 내가 원하는 예측값과 실제값의 차이

→ loss 줄면 accuracy 증가할 확률 높음.

인공 뉴런

: 사람 머릿 속 뉴런을 흉내내 나타낸 것.

  • 외부 신호들이 합침 → threshold 를 통해 다음 뉴런으로 감.

  • 외부에서 입력값 들어옴 → 더함 → 값의 역치 판단(값을 넘는지) → 남는 경우 activate → 이 활성화된 값을 다른 뉴런에 전달

인공 뉴런의 연결

뉴런 간 연결 부분 : 시냅스

정보가 저장될 수 있는 값을 만듬 ⇒ 가중치 w(곱하기)↔단일 신경망과의 차이

전달될 때 시냅스를 거쳐서 곱해서 전달됨.

인공신경망의 분류

클래스 2개 ( = 결과값 2개 )

입력 개수와 클래스 개수가 같아야함.

x1 : (2, -4), y1 : (1,0)

출력값이 (1,0)가 비슷하도록.

(1,0)이 아니지만 맞음

→ 가정적인 값이라 2가 더크면 == (1,0) ; (2,5)면 틀린 것.

입력은 데이터, 출력은 클래스 개수와 연결

인공 신경망의 학습

loss 계산

w 값 찾는 것 = 학습

gradient (미분) 사용

  • scalar : 숫자 1개
  • vector : 숫자 여러개
  • tensor : 3차원 이상
  • loss 는 scalar, label 에 가깝게 학습시키기 위해 → loss 가 줄면 label 에 가까워짐.

: 함수로 보면, (2,-4)를 넣으면 (1,0)이 나오게 하고 싶을 때, 첫번째 값이 더 크길 원하는 함수를 만드는 것.

;학습시키기 위해서 거리를 재서 거리를 줄여 f 를 학습시킴. → (8,2)를 궁긍적으로 (1,0)으로 마들고 싶은 것.

둘의 거리 = 인공신경망의 출력과 우리가 원하는 것이 얼마나 먼지 ⇒ loss

⇒ 문제점 발생

  • Loss function

    • f(x)와 y가 얼마나 가까운지 판단 (8, -2), (1,0)

      • 문제 1) scale 이 안맞음 (확률이여야 함)

        • ⇒ softmax 함수사용
          • label 이랑 계산한 거랑 비교함

      • 문제 2) 어떻게 가까운지 판단 (root mean square deviation)

        • ⇒ cross entropy 사용

      ; 두개의 확률 분포가 있을 때 그들의 거리가 어떻게 되느냐

      0 → 이상적(인공 신경망 학습시킴)

      무한 → 틀렸을 때

객체 지향 프로그래밍

: 객체를 만들어서 문제를 해결

  • 객체 : 데이터와 그 데이터에 관련되는 동작을 모두 포함. ( 공통되는 캐릭터들의 하나의 특징을 class 로 묶어놓고 객체를 만드는 것. ) ⇒ 클래스 만듦 → 필드 결정 → 객체 생성
    • 파이썬은 모두 객체 프로그래밍

  • 클래스 : 클래스 함수를 통해 만듬.

상속

: 자신 뒤 클래스에서 사용하는 것. 새로 사용하지 않고 method 와 attribute 가져옴

⇒ 자식 클래스들이 똑같은 클래스를 사용하기 위해서

사용방법 : class 클래스 이름 ( 상속 클래스 ) :

class Module(nn.module):

Numpy

  • numpy : 함수나 계산을 편리하게 하기 위한 라이브러리

    • 대표 클래스 : ndarray
    • ndarray vs list
      • list ⇒ 더하면 리스트가 직렬적으로 출력됨
        • 나누기, 더하기 실행 안됨.
      • ndarray ⇒ 배열의 index 가 맞으면 더해서 나옴.
        • scalar 연산을 하면 각각의 인덱스에 적용됨.
    • indexing : 원하는 숫자를 끄집어 내는 것.

    분류 하기위해 인공신경망 사용 인공신경망은 뉴런과 인공뉴런으로 이루어짐.

    뉴런끼리 연결되는 w값으로 이루어짐.

    인공신경망은 w값에 따라 출력이 결정.

    새로운 입력이 들어오면, 들어와서 각각의 뉴런들을 거쳐 w값을 곱하고, 곱한 값에 따라 출력이 결정됨.

    입력1 - 레이블 1 - 출력1 ⇒ 레이블과 출력이 같아지게끔. 근데 같지 않음.

    같기 위해 → w값 찾기 ⇒ loss를 가지고 (레이블과 출력의 거리) loss 작아지면 같아지는 것.

    출력에 softmas + cross entropy를 더해 새로운 loss 만듬 → 입력에 따른 새로운 loss 만들어짐.

    학습하기 위해( 인공신경망 + 알파 ) 를 붙여 입력이 되면 출력이 loss로 나옴.

    출력을 작게 만드는 인공신경망을 찾는것. 인경신경망에서 w값들이 loss를 줄이는 방식으로 학습이 되는 것.

    기존의 (w = w - w의 러닝메이트 곱하면 )이것이 학습시키는 방법
